# Get started

Requirements:
- [Docker](https://www.docker.com/)
- [Go](https://go.dev/) 1.22+

Optional:
- [Tilt](https://tilt.dev/)

## Clone starter repository
```bash
git clone https://github.com/daarlabs/starter.git
```

## Start the app
```bash
tilt up

# OR

docker compose up
```

Your app will be running on [localhost](http://localhost)
